v i n y l l y is an interactive digtal record player that visually enhances the musical listening experience with the vintage vibes of vinyl.

To listen music in vinylly, the user must verify their age (you must be 18+ to enter). Once validated, the user can enter their name (option) and a genre of music. Currently, the library is limited to a few genres (as we have no incorporated the Spotify API yet). Current genres include: rock, psychedelic, rap, hip-hop, soul, jazz, country, and pop.

CSS classes are dynamically manipulated to animate the record player and the album art (jQuery and CSS Hover libraries are used). Bands-in-Town and Last.FM APIs allow the user to view upcomming events and read more about the artist playing

This is the first generation of vinylly. There are plans to drastically improve the user experience by including a larger library (spotify API) and changing the log-in feature so that the user can directly log into their spotify account. Eventually, vinylly will allow the user to select from a variety of record player skins to customize their experience.

Videogame word guess game (basic hangman with javascript)

This is a basic word guess game powered by javascript. The user begins the game by pressing any key. Once in the game, a random (video-game themed) word populates. The number of spaces corresponds to the number of letters that must be guessed. Each time the user guesses an incorrect letter, a limb is added to the gallow. 6 wrong guess, and you're out! Be warned - this game is not very nice to you when you guess incorrect letters

This game tracks the numbers of wins and losses in a session, so you can keep track of your progress! At any point during the game, you can press the "esc" key to exit. Any keys beside letters and "esc" will be ignored!

LIRI is a Language Interpretation and Recognition Interface command line node app that takes in parameters and gives the user back data. This app utilizes multiple node.js packages, including fs, moment, chalk, axios, and spotify-node-api. The chalk package is utilized to style the output, which improves readability and enhances the user expierence.

LIRI creates and appends a log file that records commands and results. This tool is useful for troubleshooting errors received, or returning to data previously searched.
